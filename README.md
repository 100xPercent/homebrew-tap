# 100xPercent Homebrew Tap

Homebrew tap for products from [100xPercent](https://github.com/100xPercent).

## Install

```bash
brew install 100xPercent/tap/pop-pay
```

## Available Formulae

- **pop-pay** — Runtime security layer for AI agent commerce. [Upstream](https://github.com/100xPercent/pop-pay)

## Auto-bump

Formulae auto-bump on upstream release via `repository_dispatch`. Manual bump also available via the workflow dispatch UI.
